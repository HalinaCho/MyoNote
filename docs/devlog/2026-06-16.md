### 3. 전체 컬러 팔레트 교체
기존 Blue 계열 브랜드 컬러를 Teal 계열로 전면 교체

| 역할 | 기존 | 변경 |
|---|---|---|
| 브랜드 | `#3B82F6` (blue) | `#10bcad` (teal-green) |
| 성공/달성 | `#10B981` (green) | `#10bcad` (통일) |
| 경고/주의 | amber | `#fde68a` (amber-200) |
| 위험/나쁨 | red | `#fda4af` (rose-300) |
| 배경 | gray-100 | `#edf7f6` |
| 외부 배경 | — | `#d9efed` |

- 본문 텍스트 색상: teal 톤 제거 → gray 스케일 통일 (gray-800 / gray-600 / gray-400)
- 분석 탭 OS 그래프 라인: `#9CA3AF` (gray-400)
- 달성률 탭 바 차트: teal / amber / rose 3색

### 4. 홈 대시보드 디자인 개선
- 컬러 팔레트 전면 적용
- 오늘의 생활습관 카드: 상태별 컬러(teal/amber/rose) 동적 적용
- 주간 스트립 아이콘 추가: done(✓) / partial(−) / missed(✕) 세 상태 모두 아이콘 표시
- 다음 병원 예약일 날짜 옆 캘린더 아이콘 소형 배치 (헤더 아이콘 대신)

---

### 5. 기록 탭 UX 개선
- ExamTab / LifestyleTab: 상단 "기록 추가" 버튼 → 우하단 FAB(플로팅 액션 버튼) 교체
  - 반응형 고정 위치: `right: max(1rem, calc((100vw - 480px) / 2 + 1rem))`
- 생활습관 날짜별 카드 모바일 오버플로우 수정
  - `h-14` 고정 높이 제거 → `py-2.5` 유동 높이
  - 배지 `flex-1 min-w-0` 적용, `text-xs`, `truncate`로 좁은 화면 대응
- 분석 탭 빈 상태 카드: 불필요 아이콘 제거, 문구만 표시

---

### 6. 설정 탭 자녀 아바타 디자인 변경
- 기존: 컬러 채운 원형 배경
- 변경: 흰색 배경 + teal 컬러 테두리 (`border-2 border-[#10bcad]`)
- 비활성 자녀: `border-gray-200` (구분)
- "현재" 배지: 동일 스타일 (`bg-white border border-[#10bcad] text-[#10bcad]`)

### 7. 기타 버그 수정
- **FontAwesome 아이콘 깜빡임**: `layout.tsx`에 `autoAddCss = false` + CSS 정적 import로 해결
  - SSR 하이드레이션 타이밍에 동적 style 삽입 전 아이콘이 커졌다 작아지는 FOUC 문제
- **success 토스트 아이콘 색상**: 기본 초록 → `#10bcad` (브랜드 컬러 통일)
