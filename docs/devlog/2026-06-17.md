## 오늘 완료한 작업

### 1. 안축장 백분위 기능 추가 (`분석 > 안축장`)

#### 참조 데이터 및 계산 로직 (`src/lib/axialPercentile.ts` 신규)
- 동아시아 소아 코호트 기반 안축장 참조 데이터 (6~13세, P10/P25/P50/P75/P90)
  - 기존 MVP(Eyebody 프로젝트)와 동일한 수치 적용
- `calcAgeYears(birth, examDate)` — 검사 시점 나이(소수) 계산
- `calcPercentile(al, age)` — 밴드 보간법으로 백분위 산출 (P3~P97)
- `pctLabel(pct)` — 상위/하위/정상 범위 레이블 반환
- `normCurve(key)` — 차트용 스무스 참조선 포인트 배열 (나이 0.25 단위)

#### 또래 비교 차트
- X축: 나이(세), Y축: 안축장(mm) — 날짜 기반 → 나이 기반으로 전환
- P10 / P25 / P50 / P75 / P90 참조선 표시 (점선)
- P25–P75 구간 teal 채움 밴드
- 아이 실측 포인트(우안·좌안) 겹쳐 표시
- X축 범위: 오늘 기준 현재 나이 ±3세, 정수 눈금

#### 또래 비교 요약 카드
- 최근 검사 기준 OD/OS 백분위 표시
- 브랜드 teal(`#10bcad`) 컬러로 통일 — 상위/하위/정상 구분 없이 단일 색상
- PctBar: 0~100 트랙 위에 닷 마커로 백분위 위치 시각화

#### 뷰 구조 변경
- 기존: 변화 추이 / 또래 비교 토글
- 변경: 두 뷰를 한 페이지에 순서대로 통합 (토글 제거)

- **Y축 고정**: 우안·좌안 개별 토글 시에도 전체 데이터 기준 Y 범위 유지
- **커스텀 범례 토글 버튼** (Chart.js 기본 범례 제거):
  - 활성 우안: teal 테두리 + teal 점
  - 활성 좌안: gray 테두리 + gray 점
  - 비활성: 배경·테두리 제거, 점·텍스트 연한 gray로 페이드

### 4. 알림/리마인더 기능 추가

#### 알림 설정 저장 (`src/lib/notificationPrefs.ts` 신규)
- `getAlertDay()` / `setAlertDay()` — localStorage 기반 단일 알림 기준일 저장
- 설정값: 1일 전 / 3일 전 / 7일 전 중 하나 (단일 선택, 재클릭 시 해제)

#### 설정 탭 알림 섹션 (`src/app/dashboard/settings/page.tsx`)
- "알림 설정" 섹션 신규 추가 (병원 예약 알림)
- `faBell` 아이콘 + 칩 3개(1일 전 / 3일 전 / 7일 전) 단일 선택 UI
- 선택된 칩: teal 배경·테두리, 미선택: gray

#### 홈 대시보드 알림 배너 (`src/app/dashboard/page.tsx`)
- **D-day 배너**: 설정한 기준일 이내 도달 시 최상단 표시
  - D-0~D-3: rose 배경 (`bg-rose-50`)
  - D-4+: amber 배경 (`bg-amber-50`)
  - `faHospital` 아이콘, X 버튼으로 세션 내 dismiss
- **케어 미입력 배너**: 오늘 케어 기록 없을 때 표시
  - amber 배경, `faBell` 아이콘, X 버튼으로 dismiss

#### 다음 예약일 카드 D-day 배지 컬러 통일
- D-3 이하 → rose (`bg-rose-50 text-rose-500`)
- D-4~D-7 → amber (`bg-[#fde68a]/40 text-amber-600`)
- D-8 이상 → teal (`bg-teal-50 text-[#10bcad]`)
- 배너 컬러 기준과 통일
