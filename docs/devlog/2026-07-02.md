- 컬러 상징(빠름=로즈/비슷=엠버/느림=틸), 불렛, **토글**(기본 접힘). 상담 권유·불안 문구 없음(환자는 이미 진료 중). 커밋 `f89f825`~`8952583`, `a0b8c06`~`72acc1f`.

## 2. 검사 탭·입력 모달 UX 정리
- **검사 탭**: 연도 토글 **기본 전부 펼침**(`5b99aa7`). 검사 카드 하단 한 줄에 「비교」 토글(좌) + **다음 예약(우)** — 예약은 **최신 카드에만**, 미입력 시 amber '입력' 버튼→편집모달(`225631c`·`0adfff1`).
- **입력 모달 4건**(`bf01f90`): 헤더 검사일·안과 높이 통일 / 안축장 '필수'·굴절 '선택' 배지 / 굴절도수 **토글**(선택·기존값/OCR 시 자동 펼침) / 추가정보 토글 제거(항상 표시).
- **입력칸 규칙 통일**: 단일 줄 입력·업로드 버튼·SEQ 박스 모두 **h-9(36px)**. 메모만 rows 자동(`f6918bb`·`0cfe936`).
- 예시 숫자(플레이스홀더) 제거 + 입력칸 회색 배경(`bg-gray-50`)으로 흰 카드와 구분. 다음 예약일 빈칸에 `YYYY.MM.DD` 오버레이 힌트(`33d2c07`·`3d5c526`).
- 안축장·굴절 **카드 테두리 제거 → Field 스타일 통일**, 섹션 사이 **hairline 구분선**(`divide-y`)으로 경계 유지(`e9daec5`·`2804341`).

## 3. 분석탭 AI 요약 — 안전·품질
- **상담 권유 문구 → '담당 의료진' 프레이밍**(`a1740b5`): "안과 전문의와 상담" → "진단·처방은 담당 안과 의료진과 함께". 프롬프트+면책문구. 법적 안전(진단 아님) 유지 + 신규 상담 유도 없음 + 제휴 의사 강화.
- **품질 A+B**(`5276b6e`): 실천 팁을 **데이터 기반 생활습관·행동만**으로 제한(의료·상담성 금지), headline이 **지표를 엮어** 관찰하도록 규칙·스키마 정교화 + **few-shot 예시 1개**(solar-pro2는 지시보다 예시를 잘 따름).

## 4. 연간 성장률 계산 통일 (버그 수정)
문제: 「연간 성장률 카드」(전체 회귀×12)와 AI 요약(마지막 2점 환산) 수치가 달랐음.
- **공용 함수 `lib/axialGrowth.ts` 신설**, 카드+AI 공용(`19a3976`).
- 정의(유저 합의): **최근 12개월 창 검사(≥2)로 OLS 회귀×12=mm/년**. 창에 2개 미만이면 **최근 2개로 폴백**(Δ÷연수).
- **주의 2축(상호배타)**: `stale`(폴백=최근 1년 데이터 부족→전체 평균, "최근 1년 추세와 다를 수 있음"), `short`(<6개월=짧은 기간 연환산 오차 큼), `tentative`(<12개월=잠정).
- 카드 부제 `최근 1년 기준`/`최근 2회 기준`. AI엔 `annualizedNote`로 주의 취지 전달. 구식 `linearSlope`·`yearsBetween` 제거. (예측 Forecast는 미래 투영용 별도 모델이라 분리.)

## 6. OCR 안축장·굴절 OD/OS 반대 입력 개선 (커밋 `1fd6deb`)
검사지 OCR에서 우안(OD)/좌안(OS) 안축장이 종종 뒤바뀌던 문제. 원인 = 생체계측기 출력지가 통상 **OD를 왼쪽 열, OS를 오른쪽 열**에 인쇄 → 모델이 위치로 매핑하면 좌우가 바뀜.
- **추출단(근본)**: `api/exam-extract` 스키마에 공용 `EYE_RULE` 주입 — 인쇄된 눈 라벨(OD/R/우 vs OS/L/좌)로만 매칭, **열 위치·순서로 판단 금지** 명시. 안축장·굴절 전 필드 적용.
- **리뷰 UI(안전망)**: 입력 모달에 `SwapEyesButton`(좌우 바꾸기) — 안축장(axOD↔axOS) / 굴절(sphOD↔sphOS·cylOD↔cylOS 동시) 한 탭 스왑, SEQ 자동 재계산. 모델이 또 뒤집어도 재입력 없이 정정.
- `.gitignore`에 `desktop.ini` 추가.
