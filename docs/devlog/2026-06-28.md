## 0. 경쟁사 벤치마킹 (방향 설정)
서양 행동교정 앱 2종(**myopia.app/VISIONapp**, **Stop Myopia**)과 국내 판매 의료기기 **MYAH(Topcon)** 검토.
- 행동교정형(실시간 카메라 거리경고·오버레이·diopter-hours)은 **PWA 불가, 네이티브 필요** → 보류(티어3)
- myopia.app의 **진행 예측 계산기**는 웹에서 즉시 가능 + 우리는 **실측 데이터 자동 사용**(경쟁사는 수동입력)으로 차별화 → 채택
- **MYAH가 우리 방법론 검증**: 탑재 정상DB = 네덜란드 Erasmus(=Tideman 2018) + 중국 상하이 = **우리 `axialPercentile.ts` 기준선과 동일 국제표준**. MYAH=진료실 측정, 마이오노트=집에서 종단추적 → **보완재**(파일럿 영업 논리)

## 1. 진행 예측 계산기 v1
분석 탭에 **「또래·예측」 탭** 신설 + 탭 재구성.

- **탭 재구성**: `안축장/굴절도수` → **`변화추이`(안축장+굴절 추이) / `또래·예측`(백분위+예측)**
- `lib/forecast.ts`(신규): `buildForecast()` 순수계산 — (나이,안축장) **OLS 회귀로 실측 진행속도** 산출 → 시나리오 투영
- **케어 효과 역산**(핵심): 케어 중인 아이의 실측속도는 *이미 치료된* 속도 → 자연속도를 역산해야 이중계산 방지
- 효과계수 문헌값: 드림렌즈 40% / 아트로핀 50% / 병용 60% / 커스텀 40% (Ortho-K·LAMP·MiYOSMART RCT)
- `components/analytics/ForecastCard.tsx`(신규): Hero(차이)·예측곡선·연차표·가정·경고/출처, OD/OS 토글
- **굴절(D) 예측 제외**: 개인단위 신뢰도 낮음(BHVI 검증 ±0.9D·36%적중, 조절·일중변동) → 안축장(mm) 단독
- 안전장치: "측정·진단 아님, 교육용 추정" 경고문 + 논문출처 상시 노출

## 2. 고도화 — 슬라이더 / 나이별 감속 / 신뢰구간 콘
- **가정값 슬라이더**: 케어 효과(20~65%)·예측기간(1~5년) 실시간 조정 + 문헌값 리셋. `buildForecast`에 `efficacy?`/`horizon?` 오버라이드
- **나이별 진행 감속**: 직선이 아니라 또래 P50 곡선의 감속비율로 미래속도를 줄여 투영(곡선). `axialPercentile`에 `normP50`/`normSlope` 추가, 고연령 선형 폴백
- **신뢰구간 콘**: 예측선 둘레 음영 밴드(태풍 콘), 먼 미래로 갈수록 확대

## 3. 슬라이더 동작 수정 (피드백)
- 문제: 케어 효과를 바꾸면 **케어 없음(rose)** 선이 움직임 → 직관상 **케어 유지(green)**가 움직여야 함
- 수정: **케어 없음(natural) 고정**, `케어 유지 = 케어 없음 × (1−효과)` → 슬라이더가 항상 녹색 선 이동. 케어 중 아이는 문헌 기본효과로 자연속도 1회 역산(고정), 슬라이더 기본값=실측 앵커
- **신뢰구간 콘 rose 제거** → 케어 유지(teal) 음영만 표시(겹침 정리)

## 4. 짧은 측정기간 과대외삽 방지 (피드백)
- 문제: 5개월간 0.1mm를 **연 0.24mm/yr로 환산**해 추세가 가파름. 2점이면 잔차=0이라 **콘이 거짓으로 좁아** 확신처럼 보임
- 수정 ①: **6개월 미만 예측 미표시**(연환산 노이즈 증폭 차단) + 안내, 1년 미만 "잠정" 표기
- 수정 ②: **신뢰구간 콘을 기울기 표준오차(SE) 기반으로**: `halfWidth(y)=min(1.2, √(σ₀²+(slopeSE·y)²))`, `slopeSE=sReg/√Sxx`. 기간 짧거나 점 적으면 콘 자동확대, 데이터 많으면 축소 → **거짓 확신 해결**
- 검증: 5개월=숨김 / 8개월=±0.32mm(넓음) / 24개월 3점=±0.12mm(좁음)

## 5. 연간 성장률 카드 단기 주의 표기 (피드백)
- 변화추이 탭 **「연간 성장률 분석」 카드**에도 동일 기준 주의: 6개월 미만=오차 주의, 1년 미만=잠정 → 두 탭 문구·기준 통일
