## 1. 경쟁 서비스 심층 리서치
- "마이오노트와 유사한 경쟁 서비스가 있는지" 조사 — 다중 출처 웹검색 → 출처 대조 → 적대적 검증(claim 단위) 절차

### 핵심 결론 — 마이오노트의 자리는 비어 있음
1. **한국 시장 직접 경쟁(소비자용 근시관리 앱) 부재.** 국내 'MiYOSMART'(Hoya)는 근시억제 **안경렌즈(의료기기)**, 한국 사이트는 매장 안내 위주 — 앱·기록 기능 없음
2. **글로벌 클리닉 SW는 모두 자사 측정장비 종속 B2B**(ZEISS MyoConsult, OCULUS Myopia Master, Haag-Streit Lenstar Myopia) → 마이오노트의 "장비 비종속 + 부모용 PWA"가 구조적 차별점
3. 가장 가까운 소비자 앱 = 일본 **Myopia Log** (안축장 기록·그래프 전용, 기능 폭 좁음)
4. 기능 가장 닮은 **EYE is OK**(대만)는 상용 아님 — 학술 파일럿 단계
5. **결론:** "부모대면 + 안축장·굴절 추적 + 생활습관 + 케어 캘린더 + 안과 B2B2C"를 한국에서 묶은 서비스 미확인 → 차별화 여지 큼
