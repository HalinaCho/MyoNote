### 1. 카카오 OAuth 로그인 수정
- **문제**: `.env.local`의 `NEXT_PUBLIC_SUPABASE_ANON_KEY` 값이 잘못 복사돼 `Invalid API key` 오류 발생
- **해결**: Supabase 대시보드 → Settings → API에서 anon key 재복사 후 교체
- 디버그용 에러 메시지를 URL에 노출하는 임시 코드 추가 후 원인 확인, 정리 완료

### 5. 검사 기록 기능 개편

#### DB 마이그레이션 (실행 완료)
```sql
ALTER TABLE public.eyebody_exam_records
  ADD COLUMN IF NOT EXISTS sph_od numeric,
  ADD COLUMN IF NOT EXISTS sph_os numeric,
  ADD COLUMN IF NOT EXISTS cyl_od numeric,
  ADD COLUMN IF NOT EXISTS cyl_os numeric;
```

#### 굴절 도수 입력 방식 변경
- 기존: SER 직접 입력
- 변경: Sph + Cyl 입력 → SEQ 자동계산 (`SEQ = -(Sph + Cyl/2)`)
- `NegInput` 컴포넌트: `−` 부호 고정 표시, 숫자만 입력
- "조절마비 검사 결과 우선" 안내 뱃지 추가
- `types/index.ts`, `queries.ts` 업데이트

#### 검사 기록 수정 기능 추가
- 기록 카드에 ✏ 버튼 추가 (✕ 좌측)
- `updateExam` 함수 신규 구현 (`queries.ts` + `ChildContext`)
- 수정 모달에 기존 값 자동 채움 (Sph/Cyl 절댓값으로 변환하여 표시)
