## 오늘 완료한 작업

### 1. 보호자 목록 표시 및 제거 기능 (설정 탭)

#### Supabase RPC 함수 (`get_child_guardians`)
- `SECURITY DEFINER` 함수로 `auth.users`에서 이름/이메일 조회
- 호출자가 해당 자녀의 보호자인지 내부 검증
- 역할 정렬: owner → editor → viewer 순
- 표시 이름 우선순위: `full_name` → `name` → `preferred_username` → 이메일 앞부분

#### DELETE RLS 정책 추가 (`eyebody_child_guardians`)
- 소유자: 비소유자 보호자 제거 가능
- 본인: 소유자가 아닌 자신의 기록 삭제 가능 (나가기)

#### queries.ts 신규 함수
- `fetchGuardians(childId)` — RPC 호출, Guardian[] 반환
- `removeGuardian(childId, userId)` — 보호자 레코드 삭제

#### 설정 탭 UI (`src/app/dashboard/settings/page.tsx`)
- 보호자 섹션에 목록 표시 (로딩 중 animate-pulse 스켈레톤 2행)
- 역할 배지: 소유자(teal) / 편집자(amber) / 열람자(gray)
- "나" 표시: 현재 로그인 사용자 강조
- 소유자 → 비소유자에 "제거" 버튼 표시
- 비소유자 본인 → "나가기" 버튼 표시
- 제거/나가기 후 목록 즉시 갱신

### 2. PWA 설정

#### manifest.json (`public/manifest.json`)
- `name`: 마이오노트, `short_name`: 마이오노트
- `start_url` / `scope`: `/MyoNote/`
- `display`: standalone, `orientation`: portrait
- `background_color`: `#edf7f6`, `theme_color`: `#10bcad`
- 아이콘: `icon-192.png` / `icon-512.png` (purpose: "any")

#### Service Worker (`public/sw.js`)
- 캐시 전략 3종 분리:
  - Supabase API (`*.supabase.co`): 네트워크 전용 (캐시 안 함)
  - Next.js 정적 청크 (`/_next/static/`): 캐시 우선 (콘텐츠 해시 → 영구 캐시)
  - 앱 페이지: 네트워크 우선, 실패 시 캐시 폴백
- `install` 시 `skipWaiting()`, `activate` 시 이전 캐시 삭제 + `clients.claim()`

#### SwRegister 컴포넌트 (`src/components/SwRegister.tsx`)
- 클라이언트 컴포넌트, SSR 안전
- `navigator.serviceWorker.register('/MyoNote/sw.js', { scope: '/MyoNote/' })`

#### layout.tsx 업데이트
- `metadata.manifest`: manifest.json 경로 추가
- `metadata.appleWebApp`: capable/statusBarStyle/title 설정 (iOS)
- `icons.apple`: icon.png → apple-touch-icon 적용
- `<SwRegister />` body 하단 삽입

#### PWA 아이콘 교체
- `MyoNote Icon(mint,512x512).png` → 라운드 모서리(r=90, 약 17.5%) 처리
- PowerShell `System.Drawing` 으로 투명 배경 + 라운드 클리핑
- `public/icon-512.png` (512×512), `public/icon-192.png` (192×192) 생성

## 2차 세션 추가 작업

### 6. 분석 탭 차트 전반 UI 개선 (AxialTab / SerTab)

#### 생활습관 막대 차트 (LifestyleTab / LifestyleMonthlyTab)
- 목표 가이드라인을 막대 뒤로 배치 (z-index 제거, DOM 순서 활용)
- 스마트폰(목표이하 좋음) / 야외활동(목표이상 좋음) 방향에 따라 좋음구역 teal-50/60 배경 표시
- 막대 탭 시 `fmtH()` 포맷 시간 툴팁 표시 (e.g. "1시간 30분")

#### 검사기록 연도 드롭다운 (records/page.tsx)
- 캘린더 위에 연도 드롭박스 추가
- 데이터가 있는 연도만 표시, 최신 연도 기본 선택

#### 안축장/굴절도수 차트 개선 (AxialTab / SerTab)
- OD 컬러 `#10bcad` 통일 (line, dot, fill, toggle 버튼)
- OS fill 추가 (`rgba(156,163,175,0.08)`, `fill: true`, Filler 플러그인 등록)
- 차트 높이: `aspectRatio: '2/1'` CSS 방식으로 반응형 적용
- 스크롤 버튼 추가 후 → 사용자 피드백으로 제거, `useEffect([exams.length])` + `requestAnimationFrame` 으로 최신 데이터 기준 스크롤
- 또래 안축장 백분위: P 설명 레전드 아래로 이동, 차트 레전드 가운데 정렬
- PctSummaryCard를 PctView 카드 안 인라인 콜아웃으로 병합 ("최근 검사 기준")
- 굴절도수 탭 제목 "굴절도수 변화 추이"로 변경

#### Y축 단위 레이블 분리 (AxialTab / SerTab)
- TrendView: Y축 tick에서 단위 제거 → Y축 div 상단 `position:absolute` badge로 표시
  - AxialTab: `mm` badge, tick은 숫자만
  - SerTab: `D` badge 제거 후 tick에 `-1.5D` 형식 복원
- Y축 너비 52 → 44px (afterFit 적용)
- `stepSize: 0.5`, `includeBounds: false` — 0.5mm 간격, min/max bound tick 제거
- 최상위 tick null 반환 — badge와 겹침 방지
- PctView: Y축/X축 title 제거 → mm badge, `border: { display: false }` 로 chart.js v4 축 경계선 제거

#### 자녀 수정 모달 성별 flash 버그 수정 (ChildFormModal)
- `useState(EMPTY)` → `useState(editing ? {...} : EMPTY)` 초기화 방식 변경
- useEffect에서 gender 보정 시 렌더 1회 깜빡임 제거

### 7. 검사 페이지 UI 개선 (records/page.tsx)

- **안축장 행 하이라이트**: `bg-teal-50` + `text-teal-600/700` (SEQ와 동일 스타일)
- **SEQ 하이라이트 색 변경**: `bg-teal-50` → `bg-amber-50` (홈 대시보드 amber 계열 통일)
- **Sph/Cyl 입력 소수점 포맷**: `NegInput` — `type="number"` → `type="text" inputMode="decimal"` + `onBlur` 시 `.toFixed(2)` 포맷 ("3" → "3.00")
- **리스트 표시 소수점**: `fmtD()` 헬퍼 추가 → Sph, Cyl, SEQ 표시 모두 2자리 통일 (DB에 저장된 구버전 데이터 포함)
