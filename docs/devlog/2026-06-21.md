## 1. 온보딩 — 초대 코드 참여 동선 추가
- 빈 상태 온보딩의 teal CTA "자녀 등록하고 시작하기" 아래에 보조 동선 추가
- `초대 코드를 받으셨나요? **코드로 참여하기**` — 안내문 gray-400 + 액션 teal **텍스트** 버튼(채움 X)으로 위계 분리
  - 신규 사용자 주 동선(내 아이 직접 등록)은 1차, 초대 참여는 2차임을 시각적으로 구분
- 모달은 설정의 "코드로 참여하기"와 동일 UX/로직(`acceptInviteCode` + `refreshChildren`) 재사용 → 코드 입력 즉시 대시보드 전환
- 파일: `src/components/onboarding/OnboardingFlow.tsx`

## 2. 보호자 초대 — 코드 공유하기(Web Share API)
- 설정 → 보호자 초대 → 코드 생성 후 **"초대 메시지 공유하기"** teal 버튼 추가
- `navigator.share`로 네이티브 공유 시트 호출(카카오톡 등) → 앱 링크(OG 미리보기 카드) + 초대 코드 메시지 전송
- 받는 사람: 링크 탭 → 마이오노트 열기/설치 → 온보딩 "코드로 참여하기"로 코드 입력 (1번 동선과 맞물림)
- 미지원 환경(데스크톱 등)은 초대 메시지 전체를 클립보드 복사로 폴백
- 파일: `src/app/dashboard/settings/page.tsx`
- ※ 카카오 JS SDK(`Kakao.Share`) 맞춤 카드는 JS키·SDK·도메인 등록 필요 → 추후 업그레이드 후보로 보류

## 3. 생활습관 입력 모달 순서 변경 (홈 + 캘린더)
- "오늘의 생활습관" 입력 모달에서 **스마트폰(좌) · 야외활동(우)** 순으로 변경 (기존: 야외활동 좌 / 스마트폰 우)
- 캘린더 날짜 탭 시 뜨는 날짜 수정 모달도 동일 순서로 통일
- 파일: `src/app/dashboard/page.tsx`, `src/app/dashboard/calendar/page.tsx`

## 4. 설정 자녀관리 — 자녀칸 탭으로 기준 자녀 전환
- 자녀 관리 카드의 정보 영역(아바타+이름+나이)을 버튼화 → 탭하면 `switchChild` 호출, 헤더 우측 기준 자녀 전환
- "현재" 배지·아바타 teal 테두리가 탭한 자녀로 즉시 이동(시각 피드백), 펜·삭제 버튼은 기존 동작 유지
- 파일: `src/app/dashboard/settings/page.tsx`

## 5. (조사) PWA 홈화면 앱에서 카카오 로그인이 외부 브라우저로 빠지는 증상
